CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • Class attributes can be assigned as the selector, which allows us to add styles to certain elements only.
  • Assigning colors in CSS can be done by semantic references or by hexadecimal code.
  • The most common way to style a webpage is using an external CSS style sheet.
